#### What I care about

- **Self-hostable software.** No telemetry, no phone-home, your data stays on your machine.
- **Apache 2.0 by default.** No commercial restrictions. Fork it.
- **Privacy enforced at the source**, not bolted on after.
- **Spec-first OSS.** Every project ships an RFC-2119 `SPEC.md` so anyone can implement it in any language.
